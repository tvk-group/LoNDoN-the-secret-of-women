# London – The Secret of Women

Official coming-soon website for **London – The Secret of Women**, a beauty, cosmetics and wellness brand.

## Website

https://www.london-thesecretofwomen.com/

## Project Type

Static one-page website.

## Files

- `index.html` — main website
- `london-secret-of-women-square-banner.png` — square brand banner
- `london-secret-of-women-wide-banner.png` — wide website banner
- `vercel.json` — Vercel static deployment configuration
- `.gitignore` — Git ignore rules

## Deployment

This project can be deployed directly to Vercel from GitHub.

No build command is required.

## Recommended Vercel Settings

- Framework Preset: Other
- Build Command: leave empty
- Output Directory: leave empty
- Install Command: leave empty

## Domain

Add the custom domain in Vercel:

- `london-thesecretofwomen.com`
- `www.london-thesecretofwomen.com`

Recommended primary domain:

`https://www.london-thesecretofwomen.com/`
